-   What is a CDN?
    - Globally distributed network of servers (edge servers / PoPs).  
    - Stores cached copies of static content close to users.  
    - Reduces latency by serving from nearest edge.  
    - Offloads traffic from origin server.  
    - Provides security, speed, scalability, reliability.

-    CDN Terminology
    | Term          | Meaning                                |
    |---------------|----------------------------------------|
    | Origin Server | Main backend server                    |
    | Edge Server   | CDN node closest to user               |
    | POP           | Point of Presence (CDN location)       |
    | TTL           | Cache lifetime                         |
    | Cache Hit     | Served from CDN                        |
    | Cache Miss    | Fetched from origin                    |
    | Reverse Proxy | CDN forwarding to origin               |
